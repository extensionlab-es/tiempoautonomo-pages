Política de Privacidad - TiempoAutonomo

Última actualización: Febrero 2026

Descripcion general

TiempoAutonomo es una extensión de Chrome que ayuda a los autónomos españoles a registrar sus horas de trabajo y estimar sus obligaciones fiscales (IVA, IRPF, Modelo 303, Modelo 130 y cuota de autónomo).

Recogida de datos

TiempoAutonomo NO recoge, transmite ni comparte ningun dato del usuario.

Todos los datos introducidos por el usuario (clientes, sesiones de trabajo, configuración) se almacenan exclusivamente en el dispositivo del usuario mediante la API de almacenamiento local de Chrome (chrome.storage.local).

Datos almacenados localmente

Los siguientes datos se guardan localmente en tu dispositivo:

  • Información de clientes: Nombre, tarifa por hora, configuración fiscal (porcentajes de IVA/IRPF).
  • Sesiones de trabajo: Hora de inicio, duración, cliente asociado, descripción.
  • Preferencias del usuario: Estado del onboarding, ajustes de interfaz.

Permisos

TiempoAutonomo solicita únicamente el permiso storage, necesario para guardar tus clientes, sesiones de trabajo y configuración localmente en tu dispositivo. No se solicita ni se requiere ningun otro permiso.

Sin comunicación externa

  • No se envian datos a servidores externos.
  • No se utilizan herramientas de analítica ni tracking.
  • No se establecen cookies.
  • No se integran servicios de terceros.
  • No se requiere cuenta de usuario ni registro.

Propiedad de los datos

Tu eres el propietario y tienes control total sobre tus datos. Puedes:

  • Exportar tus datos en cualquier momento desde la pestaña "Exportar" del Dashboard (formato CSV o JSON).
  • Eliminar tus datos desinstalando la extensión o borrando el almacenamiento de Chrome para la extensión.

Retencion de datos

Los datos permanecen en tu dispositivo mientras la extensión este instalada. Desinstalar la extensión elimina permanentemente todos los datos almacenados. Los datos no se sincronizan entre dispositivos.

Cambios en esta política

Si esta politica de privacidad se actualiza, los cambios se reflejarán en este documento con una fecha actualizada. Los cambios significativos se comunicarán a través de las notas de actualización de la extensión.
