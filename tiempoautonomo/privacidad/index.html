Politica de Privacidad - TiempoAutonomo

Ultima actualizacion: Febrero 2026

Descripcion general

TiempoAutonomo es una extension de Chrome que ayuda a los autonomos espanoles a registrar sus horas de trabajo y estimar sus obligaciones fiscales (IVA, IRPF, Modelo 303, Modelo 130 y cuota de autonomo).

Recogida de datos

TiempoAutonomo NO recoge, transmite ni comparte ningun dato del usuario.

Todos los datos introducidos por el usuario (clientes, sesiones de trabajo, configuracion) se almacenan exclusivamente en el dispositivo del usuario mediante la API de almacenamiento local de Chrome (chrome.storage.local).

Datos almacenados localmente

Los siguientes datos se guardan localmente en tu dispositivo:

  • Informacion de clientes: Nombre, tarifa por hora, configuracion fiscal (porcentajes de IVA/IRPF).
  • Sesiones de trabajo: Hora de inicio, duracion, cliente asociado, descripcion.
  • Preferencias del usuario: Estado del onboarding, ajustes de interfaz.

Permisos

TiempoAutonomo solicita unicamente el permiso storage, necesario para guardar tus clientes, sesiones de trabajo y configuracion localmente en tu dispositivo. No se solicita ni se requiere ningun otro permiso.

Sin comunicacion externa

  • No se envian datos a servidores externos.
  • No se utilizan herramientas de analitica ni tracking.
  • No se establecen cookies.
  • No se integran servicios de terceros.
  • No se requiere cuenta de usuario ni registro.

Propiedad de los datos

Tu eres el propietario y tienes control total sobre tus datos. Puedes:

  • Exportar tus datos en cualquier momento desde la pestana "Exportar" del Dashboard (formato CSV o JSON).
  • Eliminar tus datos desinstalando la extension o borrando el almacenamiento de Chrome para la extension.

Retencion de datos

Los datos permanecen en tu dispositivo mientras la extension este instalada. Desinstalar la extension elimina permanentemente todos los datos almacenados. Los datos no se sincronizan entre dispositivos.

Cambios en esta politica

Si esta politica de privacidad se actualiza, los cambios se reflejaran en este documento con una fecha actualizada. Los cambios significativos se comunicaran a traves de las notas de actualizacion de la extension.
